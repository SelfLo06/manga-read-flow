# 本地数据布局

`data/local/**` 只存在于维护者机器，不进入 Git。真实漫画、人工标注、复核材料和导出结果不可因清理实验输出而删除。

| 目录 | 内容 | 默认保留策略 |
| --- | --- | --- |
| `sources/` | 用户合法持有的真实漫画原图 | 只读保护，不自动删除 |
| `datasets/` | 对照数据、benchmark、人工 GT、mask 与可复用标注 | 保护，不自动删除 |
| `runs/` | 当前或可复核实验运行 | 可按明确处置决定清理 |
| `reviews/` | 仍有现实用途的人工复核材料 | 保护，需明确裁决后删除 |
| `exports/` | 用户导出图片、ZIP 与 manifest | 保护，不自动删除 |
| `models/` | 本地模型与权重 | 仅在确认可重新取得且获准后删除 |
| `cache/` | 可重建缓存 | 可清理 |
| `restructure/` | 本次仓库重构的盘点、hash 清单与迁移日志 | 临时本地记录，关闭后可删除 |

受保护数据迁移必须执行“登记源路径 → 统计数量和大小 → 生成 SHA-256 清单 → 复制 → 重新计算并比较 → 更新引用 → 再次验证 → 删除旧副本”。禁止覆盖同名不同 hash 文件，禁止将真实素材混入 `runs/` 或 `cache/`。
