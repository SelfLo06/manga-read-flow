# 130 OCR

OCR 对 Detection 区域生成版本化日文文本、置信与区域 provenance。用户编辑创建新的 OCRResult version；active pointer 选择当前结果，并使翻译及视觉下游按依赖标记 stale。

当前有真实/合成样本与人工 GT，可用于工具验证；尚未证明自动 OCR 在产品支持范围内达到 M2 质量。M1 允许使用可信 OCR 输入验证单页视觉闭环。

拒绝覆盖原 OCR、把人工 GT 当运行时输入或将低置信静默接受。风险包括字符误识别、区域串块、reading order 和竖排文本。验证覆盖空白/多行、低置信、拒绝、人工编辑、版本复用与 stale propagation；自动 OCR Provider 和 M2 阈值待定。
