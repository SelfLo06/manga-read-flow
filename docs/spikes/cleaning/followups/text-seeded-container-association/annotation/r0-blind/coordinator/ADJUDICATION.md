## 3. 分歧处理规则

先比较语义和拓扑，再比较中心线位置。不得把两条语义不同的线简单平均。

以下情况直接进入 `GT_UNCERTAIN`，只验证 abstention，不进入精确 boundary 门禁：

- A/B 对容器数量或 same-container topology 不一致且讨论后仍无独立视觉证据裁决；
- 超过 `20%` 的 boundary centerline 无法在对方 `2 × delta` band 内匹配；
- 同一缺口存在两个以上同样合理的 virtual path；
- free-text 最大 support envelope 无法达成一致；
- text / not-text 判断仍需依赖算法输出或来源语义。

可以把局部区域标成 `GT_UNCERTAIN`，保留其他一致区域。证据不足时不得强制合并、分割或闭合。
