# Prompt Patterns

Prompt-patterns 是用于设计真实 prompts 的轻量模板。

它们不是真实 task prompts，也不应变成自定义 harness runtime、agent loop、script runner、hooks system 或 skill framework。

保持小而清晰。只有当某个项目阶段沉淀出可复用经验后再更新它们。

## 结构

- `case-studies/`：成功项目阶段中保留下来的示例。
- `design/`：未来 detailed-design prompts 可复用的模式。
- `implementation/`：implementation slice prompts 可复用的模式。
