# Prompt Patterns

Prompt-patterns are lightweight templates for designing real prompts.

They are not the real task prompts, and they should not become a custom harness runtime, agent loop, script runner, hooks system, or skill framework.

Keep them small. Update them only after a project phase reveals reusable lessons.

## Structure

- `case-studies/`: preserved examples from successful project phases.
- `design/`: reusable patterns for future detailed-design prompts.
- `implementation/`: reusable patterns for implementation slice prompts.
